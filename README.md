# 💃 Dance Shala - Landing Page

A modern, responsive, and animated landing page for **Dance Shala** — a dance academy offering Dance, Yoga, Aerobics, Fitness, Gymnastics, and Skating classes.

## 🚀 Features

- 🎥 Video/GIF autoplay background hero section
- 📸 Stylish gallery with images & videos
- 🌟 Infinite sliding testimonials with star ratings
- 📍 Embedded Google Map and contact section
- ⚡ Smooth scroll animations (AOS)
- 🌓 Clean, minimal design (Dark Mode ready if needed)
- 📱 Fully responsive on all devices

## 🛠️ Built With

- React.js
- CSS3 (custom)
- JavaScript (ES6)
- AOS.js (Animate on Scroll)
- Google Fonts & Icons

## 🧑‍💻 Getting Started

### 1. Clone the repository
```
git clone https://github.com/your-username/dance-shala-landing.git
cd dance-shala-landing
```

2. Install dependencies
```
yarn install
```

3. Start the development server
```
yarn start
```

Visit http://localhost:3000 in your browser.
